# 인공지능, 머신러닝, 딥러닝이란?

**인공지능 > 머신러닝 > 딥러닝**<br>
- 딥러닝은 머신러닝 분야 방법론중 인공신경망(Artifical Neural Networks)를 여러층 쌓아 올린 기법을 의미한다.
- 머신러닝은 인공지능(AI)의 방법론 중 하나이다.
머신러닝이란 "데이터를 이용해서 컴퓨터가 어떤 지식이나 패턴을 학습하는 것" 이다.
- 인공지능은 컴퓨터가 인간과 같은 지능적인 행동을 할 수 있게 해주는 기법들을 연구하는 분야이다.
                



### 딥러닝 알고리즘이 주목받는 이유
    1. 쉽게 빅데이터를 구할 수 있게된 환경 조성
    2. GPU를 필두로한 컴퓨팅 파워의 발전
    3. 새로운 딥러닝 알고리즘 개발

### 머신러닝 알고리즘의 3가지 분류
1. 지도 학습(Supervised Learning)<br>
    - 지도 학습은 정답 데이터가 존재하는 상황에서 학습하는 알고리즘이다.<br>
    -> 입력 데이터 x와 정답 레이블 y의 쌍(x,y)를 이용해서 학습하는 알고리즘이다.<br>
    - 예측하는 결과값이 이산적(Discrete)인 경우(y가 가질 수 있는 값이 [0,1,2 ..]와 같이 유한한 경우) 분류(classification)문제라고 한다.
    - 예측하는 결과값이 연속적(Continuous)인 경우(y가 실수인 경우) 회귀(Regression)문제라고 한다.
    - 딥러닝에서 지도 학습으로 방법론으로 주로 사용되는 구조는 CNN(Convolutional Neural Network), RNN(Recurrent Neural Network)이다.
 2. 비지도 학습(Unsupervised Learning)
    - 비지도 학습은 정답 레이블 y 없이 입력 데이터 x만을 이용해서 학습하는 알고리즘 이다.<br>
    -> 입력데이터 (x) 형태로 학습을 진행
    - 비지도 학습은 데이터의 숨겨진 특징(Hidden Feature)을 찾아내는 것이 목적이다.<br>
    -> 단독으로 사용하기보다는 파악한 데이터의 숨겨진 특징을 원본 데이터 대신 지도학습의 인풋 데이터로 활용해서 지도 학습의 성능을 끌어올리는 용도로 많이 활용한다.
    - 비지도 학습의 방법론으로는 주성분 분석(Principal Component Analysis)이 있고 딥러닝에서 많이 사용되는 구조는 오토인코더(Autoencoder)이다.
 3. 강화 학습(Reinforcement Learnig)
    - 강화 학습은 에이전트(Agent)가 주어진 환경(State)에서 어떤 행동(Action)을 취하고 이에 대한 보상(Reward)을 얻으면서 학습을 진행한다.<br>
    -> 에이전트는 보상을 최대화하도록 학습을 진행한다.
    - 강화 학습은 동적인 상태에서 데이터를 수집하는 과정까지 학습 과정에 포함되어 있는 알고리즘이다.
    - 대표적인 알고리즘으로는 Q-Learning이 있고, Q-Learning과 딥러닝을 결합한 DQN(Deep-Q-Network)기법이 있다.
